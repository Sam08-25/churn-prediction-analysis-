# Churn Prediction Analysis

## Overview
This project predicts customer churn and employee attrition using machine learning models built with LightGBM. The project includes exploratory data analysis, SHAP explainability, survival analysis, retention strategy recommendations, and a Streamlit dashboard.

## Features
- Customer churn prediction
- Employee attrition prediction
- SHAP-based model explainability
- Survival analysis
- Interactive Streamlit dashboard
- Risk simulator for churn prediction

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- LightGBM
- SHAP
- Matplotlib
- Streamlit

## Project Structure
- notebooks/ → Jupyter notebooks
- app/ → Streamlit dashboard
- models/ → trained ML models
- data/ → datasets
- outputs/ → charts and reports
